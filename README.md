## 几个实际使用中会遇到的问题

**SSH 密钥是什么情况？**

DMIT 默认不支持密码登录，只能用 SSH 密钥。第一次配置要花几分钟，官网有中文教程，跟着走一遍没问题。安全性更高，但确实不如密码方便。

**流量用完了怎么办？**

T1 系列超量不停机，直接降速继续跑（HKG T1 限速到 50Mbps），等下个计费周期恢复满速。Pro 和 EB 系列超量后服务暂停，等下个周期。个人用户如果流量消耗不算大，T1 这个机制挺友好的。

**套餐缺货怎么处理？**

DMIT 的热门套餐，尤其是年付特惠款（比如 HKG.Pro.VICTORIA、T1.WEE），经常补货后很快就没了。官网商品页会实时显示 In Stock / Sold Out，有需求的话定期刷一下，看到有货不要拖。

**支付方式有哪些？**

支持支付宝、微信支付、PayPal。对大陆用户来说，支付宝付美元很方便，汇率按实时浮动。

**IP 被墙了怎么办？**
